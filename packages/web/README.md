# 视频下载器前端

独立的前端项目，使用 Vite 7.3.1 + React 18 + TypeScript 构建。

## 技术栈

- **Vite**: 7.3.1
- **React**: 18.2.0
- **TypeScript**: 5.2.2

## 安装依赖

```bash
# 使用 npm（推荐，如果 pnpm 有问题）
npm install

# 或使用 pnpm
pnpm install
```

## 开发

```bash
pnpm run dev
```

前端开发服务器将在 http://localhost:5173 启动。

确保后端 API 服务器运行在 http://localhost:3000。

## 构建

```bash
pnpm run build
```

构建产物将输出到 `dist` 目录。

## 预览构建结果

```bash
pnpm run preview
```

## 代码检查

```bash
pnpm run lint
```
